# regdomain.5

`regdomain.xml` — 802.11 无线监管定义

## 名称

`regdomain.xml`

## 描述

`regdomain.xml` 文件描述了 IEEE 802.11 无线电波的运行监管规定。

此信息由 [ifconfig(8)](../man8/ifconfig.8.md) 程序使用，以构建用于下载到系统的监管状态。仅当监管规定发生变化时才应更改此文件。

## 文件

**/etc/regdomain.xml** — 802.11 监管约束的 XML 数据库

## 参见

[wlan(4)](../man4/wlan.4.md), [ifconfig(8)](../man8/ifconfig.8.md)
