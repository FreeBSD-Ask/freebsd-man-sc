# cd9660(4)

`cd9660` — ISO-9660 文件系统

## 名称

`cd9660`

## 概要

要链接到内核：

> options CD9660

或者，要在引导时以模块形式加载该驱动，请在 [loader.conf(5)](../man5/loader.conf.5.md) 中加入以下行：

```sh
cd9660_load="YES"
```

## 描述

`cd9660` 驱动允许 FreeBSD 内核访问 cd9660 文件系统。

## 实例

要挂载位于 **/dev/cd0** 上的 `cd9660` 卷：

```sh
mount -t cd9660 /dev/cd0 /mnt
```

## 参见

etdump(1), nmount(2), unmount(2), [cd(4)](cd.4.md), [fstab(5)](../man5/fstab.5.md), [mount(8)](../man8/mount.8.md), [mount_cd9660(8)](../man8/mount_cd9660.8.md)

## 历史

`cd9660` 驱动首次出现于 4.4BSD。
