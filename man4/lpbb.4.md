# lpbb.4

`lpbb` — 并口 I2C 位操作接口

## 名称

`lpbb`

## 概要

`device iicbus device iicbb`

`device lpbb device iic`

## 描述

*lpbb* 驱动支持 Philips 官方 I2C 并行位操作接口。

```sh
                                         LS05 pin 14 (Vcc) o      -------
                                                           |      |     |
            +--+--+---------------------+--+--+------------+------+-o 1 |
            |  |  |                     |  |  |           ===.1uF | +5V |
  -------- [R][R][R] 3x10K       3x10K [R][R][R]   LS05    |      |     |
  |      |  |  |  |                     |  |  |    pin 7 o-+------+-o 2 |
  | 12 o-+--+  |  |   3| 4             |  |  |    (Gnd)          | GND |
  | 17 o-+-----|--|----| >o-------------+--|--|--------------+    |     |
  |      |     |  |    |/        8 /|9     |  |     10 /|11  +----+-o 3 |
  | 15 o-+-----+--|--------------o< |------+--|------o< |----+    | SCL |
  |      |        |   1| 2        \         |        \         |     |
  |  9 o-+--------|----| >o-------------------+--------------+----+-o 4 |
  |      |        |    |/                            6 /|5   |    | SDA |
  | 11 o-+--------+----------------------------------o< |----+    -------
  | 10 o-+-+                                           \          4-pin
  | 13 o-+-+--oGND                                              Connector
  | 25 o-+-+      ------------------ 零件清单 --------------------------
  --------        | 1 - .1 uF 电容        | 6 - 10K 5% 电阻            |
  25 针公 D 型    | 1 - 4 针连接器        | 1 - 25 针公 D 型连接器      |
  连接器至 PC     | 1 - 74LS05 开漏集电极六反相器                       |
  打印机端口      -------------------------------------------------------
```
